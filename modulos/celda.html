Celdas Robóticas

Los procesos en los que se van a usar celdas robotizadas son soldadura, limpieza de soldadura y pintura. La idea es que las 3 sean celdas basadas en robots. Esto con el fin de lograr que se reduzcan los tiempos de proceso, que haya un mejor flujo de materiales y bajar el riesgo de accidentes. Para lograr un diseño óptimo hay que entender el proceso que se va a realizar en la celda robotizada, determinar las acciones puntuales que se van a llevar a cabo.

  • Robot de 6 DOF
  • Herramienta de pintura para el robot
  • Bandas transportadoras
  • Controlador para el robot
  • Controlador para las bandas transportadoras
  • Sensores de posición para las bandas transportadoras
  • Elementos de seguridad

Para simular la celda robotizada para pintura se usó RobotStudio. Aquí se pueden crear elementos como robots, sensores, controladores, cuerpos rígidos y objetos que interactúan con el robot. Además se pueden definir las rutas que seguirá el robot y el software las convierte a código RAPID. En nuestro caso se hizo la secuencia de control para que el robot pinte una silla, donde se incluye el giro de la silla para poder pintarla por ambos lados, bandas transportadores y botones que reemplazan ciertas acciones del operario.
