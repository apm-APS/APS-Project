Industria 4.0

Para entender qué es la automatización es importante conocer la norma estándar ISA 95. En esta se explican los 5 niveles de la pirámide de automatización, todos importantes para lograr automatizar procesos. Entender y aplicar los conceptos y métodos de todos los niveles (en especial los 3 primeros) será el objetivo principal del proyecto integrador.

Primer Nivel

El primer nivel es el de campo, donde van los sensores y actuadores que tiene la planta. Por ejemplo robots, CNCs, bandas transportadoras, etc.

Segundo Nivel

El segundo nivel es el de control. Desde este nivel se controlan las acciones y variables de todo lo que está en el primer nivel. Para esto usualmente se usan PLC, DCS o PAC.

Tercer Nivel

El tercer nivel es el de supervisión, donde se hace adquisición de datos, supervisión, operación, alarmar y se guardan los datos históricos. Para esto se usan sistemas SCADA

Cuarto Nivel

El cuarto nivel es el de planeación, aquí se hace la gestión de producción con sistemas MES (Manufacturing Execution System).

Quinto Nivel

El quinto nivel es el de gerencia, aquí se hace la gestión corporativa utilizando sistemas ERP (Enterprise Resources Planning).

Implementación Industria 4.0.

Los principales aspectos de la industria 4.0 son las técnicas avanzadas para producción, las nuevas tecnologías y la integración de estas con las personas. Esto incluye la robótica, inteligencia artificial, internet de las cosas, digitalización entre otros. Para este proyecto se tendrá en cuenta la robótica, la digitalización, el acceso a datos y el uso de nuevas tecnologías para fabricación. Los distintos niveles dependen de los datos que adquieren de los otros, es por esto que la comunicación entre estos es muy importante. En principio, la norma ISA 95 tiene comunicación bidireccional entre dos niveles consecutivos como se ve en la imagen:


Pero esto puede llegar a ser un problema pues es muy vertical, y si un nivel deja de funcionar, todos pueden dejar de hacerlo también. Por esto es importante el paso a la industria 4.0, donde se busca que máquinas, sistemas, sensores y hasta personas estén interconectados. Para lograr esto se puede usar el internet de las cosas (IoT) para que toda la información esté en la nube y sea más fácil saber qué pasó en caso de un inconveniente y poder resolverlo. Implementar esto en el proyecto sería una etapa futura, para poder tener datos de los distintos niveles de la pirámide de automatización, en tiempo real, donde sea y cuando se necesite.
