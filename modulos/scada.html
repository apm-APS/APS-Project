Sistema de Supervisión SCADA

El término SCADA aplica para dos formas de control supervisorio. Primero como Sistema donde se diseña un sistema que se implementa en superficies geográficas amplias utilizando telemetría, adquisición de datos remotos, RTUs, MTUs, etc. Segundo como Software para hacer control supervisorio basado en PC.
Para este proyecto se diseñará un Software SCADA ya que se busca supervisar la fabricación de sillas en un espacio pequeño por lo que no es necesario el diseño de este como Sistema. Se hicieron 2 HMI, una específica para la celda robotizada del proceso de pintura y otra para la fábrica digital. El diseño de ambas se hizo en Softwares distintos para explorar distintos métodos de diseñar una HMI. La de la celda robotizada se hizo utilizando el complemento FlexPendant de RobotStudio, el cual permite crear interfaces a través de ScreenMaker y se validó su funcionamiento con una simulación de la HMI en conjunto con la celda robotizada. Para lo que no son celdas robotizadas, se utilizó Ignition para desarrollar la HMI, supervisando la conexión entre RSLinx Classic (Studio 5000) y Siemens NX. Para validar su funcionamiento fue necesario recurrir a las conexiones de comunicación hechas anteriormente para que los programas funcionaran en conjunto.

Diseño de HMI celda robotizada

En la pantalla principal se le avisa al operario el estado de la silla que se está pintando, si es necesario moverla, si es necesario recargar el tanque de pintura, la producción diaria de sillas y si se cumplió la meta de producción. Además de contar con botones para iniciar la producción e indicar si se puede continuar con esta después de las correcciones que se necesiten hacer y hay un botón para acceder a la segunda pantalla. En la segunda pantalla se ve el estado del robot, en este caso, la posición de las articulaciones del robot. Para lograr esto se necesita conectar el controlador del robot con el software de supervisión.

Validación celda robotizada

RobotStudio permite conectar el FlexPendant con el controlador de la estación de la celda robótica. De esta manera se puede tener un HMI hecho con FlexPendant para supervisar y controlar algunos elementos del robot, como los son los ángulos de sus 6 ejes, el nivel de pintura, los números de ciclos que ha realizado, así como indicar si el robot ya terminó su ciclo y está listo para el siguiente ciclo, o si ya es necesario girar la silla para que la pinte por detrás; de esta manera se cuenta también con botones para indicar que ya se puso la silla (Posición 1) o si ya se giró la silla (Posición 2) para que el robot siga con su respectivo proceso. También indica si hay colisiones.




Diseño de HMI fábrica digital

Para la interfaz de supervisión de la fábricca digital se tuvieron en cuenta las 3 máquinas que se simularon. Primero, la sierra para el corte de los perfiles, que como es un proceso prácticamente manual lo que se puede supervisar es la cantidad de perfiles cortados, esto se representa con una barra de progreso la cuál también tiene un cuadro de texto que indica el número de perfiles que se espera cortar en el día. Segundo, la troqueladora que al igual que la sierra es un proceso manual así que se presentan los mismos valores. Tercero, el brazo robótico en el que se presentan otras variables además del número de sillas pintadas, como el nivel de pintura (para saber cuándo se debe recargar). Las 3 tienen un indicador de estado para saber si están encendidas, apagadas o si fallaron. Además se presenta un indicador de producción para saber si la fabricación de sillas va a la "velocidad" esperada.
